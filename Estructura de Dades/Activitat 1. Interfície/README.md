# Activitat 1: Implementació d'una interfície

*Aquesta activitat és obligatòria i comptarà un 5% de la nota de pràctiques de l'avaluació contínua*

En aquesta activitat implementarem una classe que compleixi amb les funcions descrites a una interfície, i posteriorment la farem servir per a emmagatzemar les dades corresponents a un llistat de pel.lícules.

Les tasques que heu de realitzar son les següents:

- Programar una classe que implementi l'interfície LlistaGenerica, i que guardi les dades fent servir una ArrayList (https://docs.oracle.com/javase/8/docs/api/java/util/ArrayList.html)

- Programar tots els mètodes del Main.java per a carregar les dades del fitxer movies.txt i els mètodes per a poder treballar amb aquestes dades. Tingueu en compte el tractament de les excepcions per a aquells casos on faci falta.
